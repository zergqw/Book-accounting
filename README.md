# Book-accounting
Учёт книг в десктоп-приложении на Electron.

## Функционал
- добавление, удаление, редактирование, поиск книг

## Технологии
- Frontend: Electron, JS
- Backend: Node.js
- БД: PostgreSQL
- Тесты: Jest

## Установка и использование
- npm i 
- npm run migrate
- npm run start
